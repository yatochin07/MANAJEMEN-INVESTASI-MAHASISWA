# EduVesting — Student Investment Management Dashboard

EduVesting is a fullstack web dashboard designed for university students to manage and simulate investment portfolios digitally.

This project was built as a modern fintech-style dashboard with portfolio analytics, transaction tracking, AI insights, financial goal planning, and asset allocation management.

---

# Features

## Authentication
- Login system
- Register system
- Session using localStorage

## Dashboard
- Portfolio summary
- Financial metrics
- Asset allocation overview
- Smart recommendation section

## Portfolio Management
- Multi-asset portfolio
- Stock, crypto, gold, and mutual fund tracking
- Profit/Loss monitoring

## Transactions
- Transaction history
- Buy/Sell simulation
- Investment activity tracking

## Financial Goals
- Goal tracking system
- Progress calculation
- Financial planning cards

## AI Insights
- Simulated AI financial recommendation system
- Portfolio analysis
- Risk and diversification suggestions

## Asset Allocation
- Portfolio allocation visualization
- Asset percentage analysis

## Fee Calculator
- Investment fee simulation
- Buy/Sell fee estimation
- Multi-instrument support

## Settings
- User profile management
- Dynamic username synchronization
